*markdown-front-matter.txt*  Documentation for Markdown Front Matter plugin

==============================================================================
FEATURES                                         *markdown-front-matter-features*

- **Generate Front Matter**: Generate YAML front matter for Markdown documents.
- **Generate metadata by llm**: Use llm to generate metadata(title, description, categories, tags)
- **Intelligent Formatting**:
  - Auto-generated slugs in kebab-case from filenames
  - ISO 8601 compliant dates with timezone information

==============================================================================
INSTALLATION                                     *markdown-front-matter-installation*

1. Via package manager (recommended):

```lua
-- packer.nvim
use {
  'gitsang/markdown-front-matter.nvim',
  opts = {},
}

-- lazy.nvim
{
  'gitsang/markdown-front-matter.nvim',
  opts = {},
}
```

```lua

  }

```

==============================================================================
COMMANDS                                         *markdown-front-matter-commands*

:MarkdownFrontMatter
    Updates the front matter in the current Markdown file or creates it if it 
    doesn't exist. Includes the following fields:
    - title: Title (default empty, generated by llm)
    - slug: Generated in kebab-case format based on the filename
    - description: Description (default empty, generated by llm)
    - date: Current time if not set (ISO 8601 format, +08:00 timezone)
    - lastmod: Last modification time (same format as date)
    - weight: Weight (default 1)
    - categories: Categories (empty array by default, generated by llm)
    - tags: Tags (empty array by default, generated by llm)

==============================================================================
RULES                                            *markdown-front-matter-rules*

1. Automatic slug generation rules:
   - Converts to lowercase
   - Replaces spaces and underscores with hyphens
   - Removes non-alphanumeric characters
   - Merges consecutive hyphens
   - Trims leading and trailing hyphens

2. Time format:
   - Automatically generates timestamps compliant with ISO 8601 standard
   - Includes +08:00 timezone information
   - Precise to the second

3. Front Matter Detection:
   - Automatically detects existing front matter (between --- delimiters)
   - Updates existing front matter preserving its structure
